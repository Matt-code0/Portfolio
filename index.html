About this Portfolio

I created this portfolio to show what I have learned during my intensive IT course, and I'm still working on it! These are not just exercises – each project is designed to work like a real application. This portfolio helps me track my progress and demonstrates the skills I am building as a junior developer.

My main focus is on C# and ASP.NET Core, but I also enjoy playing with the frontend to make things look good. Every project here has a link to GitHub where you can check my code, and a live demo so you can see it in action. I try to write clean code and learn something new with every commit.

  • Real Projects: No more "Hello World" – these are functional web apps.
  • Learning by Doing: I focus on backend logic, databases, and responsive design.
  • Open Source: You can see my entire journey and all my mistakes on GitHub.
  • Active Growth: I'm currently finishing the course and looking for my first junior role.
