About this Portfolio

I created this portfolio to showcase my journey through an intensive IT course. The projects here are not just simple exercises, but functional applications built to solve practical problems. This portfolio helps me track my progress as I explore different technologies and build a solid foundation as a junior developer.

The course provided a broad introduction to full-stack development, covering backend logic, databases, and responsive frontend design. I enjoy learning how different parts of a system work together and improving my understanding through hands-on projects. Each project includes a GitHub repository where my learning process and progress are documented.

  • Full-Stack Exposure: Experience with both backend concepts and modern web interfaces.
  • Practical Projects: Functional applications beyond basic examples.
  • Learning in Practice: Skills developed through real project work during an intensive course.
  • Junior Progress: Currently finishing my studies and preparing for a first junior role.
